# Diagrams

Pasta para diagramas que suportam a explicacao do laboratorio.

## Itens Recomendados

1. Topologia do cluster local (control-plane + workers).
2. Fluxo de scheduling com `nodeSelector`, affinity e anti-affinity.
3. Fluxo de decisao do HPA (metricas -> replicas alvo).
4. Impacto de taints/tolerations na distribuicao de pods.

## Formatos

- Fonte editavel: `.drawio` (ou equivalente)
- Export para leitura rapida: `.png` ou `.svg`
